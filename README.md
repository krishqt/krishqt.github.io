# krishqt.github.io

## Useful Python Tool

`useful_tool.py` generates a quick summary report for a directory, including file counts,
extension breakdowns, total size, and optional tree and largest-file views.

### Usage

```bash
python useful_tool.py
python useful_tool.py /path/to/dir --top 10 --tree --max-depth 3
python useful_tool.py /path/to/dir --include-hidden --top 0
```
